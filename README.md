# Startpage NoScript dark theme 
Startpage dark theme for [NoScript](https://noscript.net/) that support also Videos and News categories. Works with [Startpage](https://addons.mozilla.org/en-US/firefox/addon/startpage-private-search/?utm_source=addons.mozilla.org&utm_medium=referral&utm_content=search) Firefox extension. Made for Firefox extension [Stylus](https://add0n.com/stylus.html).
Hosted in [userstyles.world](https://userstyles.world/style/26138)
